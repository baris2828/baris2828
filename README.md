### 🚀 Featured Projects

#### [CLT-Real-World-Analysis](https://github.com/baris2828/CLT-Real-World-Analysis)
A deep-dive into the **Central Limit Theorem**.
* **The Goal:** Mathematically proving that sampling distributions follow a normal curve, even when the source data (e.g., income) is highly skewed.
* **Tech Stack:** Python, Pandas, Seaborn, Pytest.
* **Quality First:** 100% automated test coverage for statistical logic.

#### [eniac-ab-test-optimization](https://github.com/baris2828/eniac-ab-test-optimization)
A four-arm **A/B test** analysis for an e-commerce homepage CTA button.
* **The Goal:** Identify the highest-performing CTA variant while controlling the family-wise error rate.
* **Methods:** Chi-Square test of independence, Bonferroni-corrected pairwise post-hoc tests, relative-lift framing.
* **Tech Stack:** Python, pandas, SciPy, Matplotlib, Seaborn, Jupyter.

#### [predictive-maintenance-dashboard](https://github.com/baris2828/predictive-maintenance-dashboard)
End-to-end analytics case study for a simulated German injection-molding factory.
* **The Goal:** Move from reactive to predictive maintenance — surface a leading sensor metric so the shop-floor team can intervene *before* failures occur.
* **Key Insight:** When vibration exceeds 25 mm/s, downtime likelihood rises roughly **60× versus the nominal range** — a single actionable threshold, no ML required.
* **Tech Stack:** Python (pandas, NumPy) for synthetic data, Power BI for modeling and dashboards, DAX for KPIs (scrap rate, downtime %, OEE proxy).
